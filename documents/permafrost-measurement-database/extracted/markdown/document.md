## Overview

This document contains structured performance measurement data from 254 HVAC system tests before and after PermaFrost NMR treatment. All measurements follow the International Performance Measurement and Verification Protocol (IPMVP).

## Data Fields

Each measurement record contains:

### Equipment Information
- Equipment type, manufacturer, model
- Refrigerant type
- Rated capacity (tons)
- Unit age

### Baseline Test Measurements
- Electrical: Voltage, Amperage, Calculated Watts
- Pressures: Suction and Discharge (pressure and temperature)
- Air Temperatures: Entering/Leaving (dry bulb and wet bulb)
- Performance: Airflow (CFM/GPM), Capacity (tons), Enthalpy delta
- Efficiency: kW/ton, EER, COP

### Post-Treatment Measurements
- Same parameters measured after PermaFrost NMR treatment
- Tests conducted under comparable operating conditions

### Calculated Results
- Efficiency change percentage
- Energy saved (kWh)
- Cost savings ($)
