       Detailed Results for Global IT Services Provider Chiller
System Description / Design Characteristics
The pilot chiller provides cooling to the Global IT Services Provider Campus. Given the internal load of the facility and 
the lifetime run hours on the control panel, it is estimated that the system runs a minimum of ~4,000 hours 
or more each per year prior to the ECM being implemented.
